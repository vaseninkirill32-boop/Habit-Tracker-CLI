# 📊 Habit Tracker CLI

Консольное приложение на Python для управления привычками, отслеживания прогресса и анализа личной статистики.

Проект создан для практики работы с логикой приложения, хранением данных, пользовательским интерфейсом в консоли и построением системы статистики.

---

## 📌 Возможности

### 📝 Управление привычками
- Добавление новых привычек
- Удаление существующих привычек
- Просмотр списка всех привычек

### ✅ Отслеживание выполнения
- Отметка выполненных привычек за текущий день
- Отмена ошибочных отметок
- Автоматический подсчет прогресса

### 🔥 Система серий (Streaks)
- Подсчет количества дней непрерывного выполнения
- Сброс серии при пропуске привычки
- Отображение текущих результатов

### 📈 Статистика
- Прогресс выполнения привычек за день
- Рейтинг привычек по длительности серии
- Система достижений:
  - 🥇 1 место
  - 🥈 2 место
  - 🥉 3 место

---

## 🛠️ Технологии

- Python 3.x
- Работа с файлами и данными
- Консольный интерфейс (CLI)
- Алгоритмы подсчета статистики
- Стандартная библиотека Python

Дополнительные библиотеки не используются.
